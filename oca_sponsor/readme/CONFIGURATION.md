
To add new Sponsor Level:
* if the module `website_crm_partner_assign` is installed (which is not needed anymore):
  browse to the menu *CRM / Configuration / § Resellers / Partners Level*
* if not, there is no entry menu for Sponsor Levels. However, you may manage them from
  any Contact, opening the *Sponsorship* tab and clicking on *Sponsor Level* field.
  You may edit existing or add new Sponsors levels from here.
